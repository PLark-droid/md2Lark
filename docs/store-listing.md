# md2Lark - Chrome Web Store Listing

## English

### Short Description (132 chars max)
Convert Markdown to Lark-optimized format. Perfect paste from AI tools (Claude, ChatGPT) into Lark/Feishu docs.

### Detailed Description

**md2Lark** converts Markdown to Lark (Feishu)-optimized HTML for perfect clipboard paste.

**The Problem:**
When you copy AI-generated Markdown (from Claude, ChatGPT, etc.) and paste it into Lark, formatting breaks — headings collapse, code blocks lose syntax highlighting, tables misalign, and nested lists flatten.

**The Solution:**
md2Lark converts Markdown into Lark-optimized HTML with inline styles, so everything pastes perfectly.

**Features:**
- Real-time preview with split-pane editor
- One-click "Fetch from AI" to grab content from Claude or ChatGPT pages
- Keyboard shortcut (Ctrl+Shift+L) to convert selected text on any page
- 3 style templates: Minimal, Enhanced, Document
- Conversion history (50 entries) with quick reload
- Full settings page for customization
- Dark mode support
- 100% local processing — no data sent to servers

**Supported Markdown:**
- Headings, bold, italic, strikethrough
- Code blocks with language labels
- Tables with proper borders
- Ordered and unordered lists (nested)
- Blockquotes
- Links and images
- Task lists (checkbox → emoji)
- LaTeX math (→ code fallback)
- Footnotes (→ inline expansion)

**Privacy:** All conversion happens locally in your browser. No data collection, no external APIs, no tracking.

## 日本語

### Short Description
MarkdownをLark最適化形式に変換。AI（Claude, ChatGPT）からLarkへの完璧な貼り付けを実現。

### Detailed Description

**md2Lark**は、MarkdownをLark（Feishu）最適化HTMLに変換し、クリップボードからの完璧な貼り付けを実現するChrome拡張機能です。

**課題：**
AIが生成したMarkdown（Claude、ChatGPTなど）をコピーしてLarkに貼り付けると、フォーマットが崩れます。見出しが消え、コードブロックのシンタックスハイライトが失われ、テーブルがずれ、ネストされたリストが平坦化されます。

**解決策：**
md2LarkはMarkdownをインラインスタイル付きのLark最適化HTMLに変換し、完璧な貼り付けを実現します。

**主な機能：**
- リアルタイムプレビュー付きスプリットペインエディタ
- ワンクリック「Fetch from AI」でClaude/ChatGPTページからコンテンツを取得
- キーボードショートカット（Ctrl+Shift+L）で選択テキストを変換
- 3つのスタイルテンプレート：Minimal、Enhanced、Document
- 変換履歴（50件）のクイック再読み込み
- カスタマイズ用設定画面
- ダークモード対応
- 100%ローカル処理 — データはサーバーに送信されません

**プライバシー：** すべての変換はブラウザ内でローカルに実行されます。データ収集、外部API通信、トラッキングは一切ありません。
